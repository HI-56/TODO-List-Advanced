React Todo List Project

A simple Todo List application built with React. This project helps beginners understand React concepts like state management, events, conditional rendering, and using localStorage to persist data.

Features

Add Tasks – Users can add new tasks to the list.

Mark Tasks as Done – Click a task to mark it as completed.

Delete Tasks – Remove a task from the list.

Filter Tasks – View Completed or Not Completed tasks separately.

Persistent Storage – Tasks are saved in localStorage, so data remains after page refresh.

Technologies Used

React – For building the user interface.

JavaScript (ES6+) – Core programming logic.

CSS / Material UI – For styling components and UI elements.

localStorage – To store tasks persistently.

Node.js and npm installed

Basic knowledge of React

Installation
# Clone the repository
git clone <your-repo-url>

# Navigate to project folder
cd react-todo-list

# Install dependencies
npm install

# Start the development server
npm start

Open http://localhost:3000
 to see the app in the browser.

Usage

Type a task in the input box and click Add.

Click on a task to mark it as done (it will change color).

Click the delete button (🗑️) to remove a task.

Refresh the page — tasks will remain because they are saved in localStorage.

Learning Outcomes

Understanding React hooks (useState, useEffect)

Managing arrays in React state

Conditional rendering (done / not done tasks)

Working with localStorage

Basic project structure and component-based architecture
